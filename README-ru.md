# 🧮 Тренажёр таблицы умножения

Простая программа для проверки знания таблицы умножения. Откройте `mtt.html` в браузере и проверьте себя.

## Как пользоваться

1. Откройте `mtt.html` в любом современном браузере.
2. На экране **Home** отметьте галочками числа, которые хотите потренировать (по умолчанию выбраны все от 2 до 9).
3. Нажмите **«Начать тест»**.
4. На экране **Test** вы увидите пример вида `7 × 8 = ?`. Введите ответ в поле и нажмите **Enter**.
   - ✅ Если ответ правильный — строка станет зелёной, и появится следующий пример.
   - ❌ Если ответ неправильный — строка станет красной, покажется правильный ответ, и тот же вопрос будет задан снова.
5. Справа сверху отображается сетка 8×8, где:
   - 🟢 зелёные клетки — примеры решены правильно
   - 🔴 красные — были ошибки (ожидают перепроверки)
   - ⚪ белые — ещё не задавались
   - ⬜ серые — числа не выбраны для проверки
6. Когда все примеры будут решены правильно, появится сообщение **«Таблица умножения проверена»**.

Программа сама запоминает, какие примеры вы уже решили, так что можно закрыть страницу и продолжить позже.
