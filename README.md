# 🧮 Multiplication Table Trainer

A simple program to test your multiplication table knowledge. Open `mtt.html` in a browser and test yourself.

## How to use

1. Open [mtt.html](https://pilat66.github.io/multiplication-table-trainer/mtt.html) in any modern browser.
2. On the **Home** screen, check the numbers you want to practice (all numbers from 2 to 9 are selected by default).
3. Click **«Начать тест»** (Start test).
4. On the **Test** screen you will see an example like `7 × 8 = ?`. Type your answer in the field and press **Enter**.
   - ✅ If the answer is correct — the line turns green, and the next example appears.
   - ❌ If the answer is wrong — the line turns red, the correct answer is shown, and the same question is asked again.
5. The 8×8 grid on the right side shows:
   - 🟢 green cells — examples answered correctly
   - 🔴 red cells — had mistakes (awaiting recheck)
   - ⚪ white cells — haven't been asked yet
   - ⬜ gray cells — numbers not selected for testing
6. When all examples are answered correctly, the message **«Таблица умножения проверена»** (Multiplication table verified) appears.

The program remembers which examples you've already solved, so you can close the page and continue later.
